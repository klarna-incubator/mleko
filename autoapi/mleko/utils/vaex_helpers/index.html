mleko.utils.vaex_helpers.get_columns(df: vaex.DataFrame, columns: list[str]) → vaex.DataFrame#

Get specified columns from a DataFrame.

Parameters:

  • df (vaex.DataFrame) – The input DataFrame.

  • columns (list[str]) – A list of the names of the desired columns or an Expression.

Returns:

A DataFrame containing only the specified columns.

Return type:

vaex.DataFrame

Examples

>>> import vaex
>>> from mleko.utils import get_columns
>>> df = vaex.from_arrays(column1=[1, 2, 3], column2=[4, 5, 6], column3=[7, 8, 9])
>>> get_columns(df, ["column1", "column3"]).get_column_names()
['column1', 'column3']
mleko.utils.vaex_helpers.get_filtered_df(df: vaex.DataFrame, filter: vaex.Expression) → vaex.DataFrame#

Get filtered DataFrame.

Will return a DataFrame containing only the rows that satisfy the filter. The filter is an Expression that evaluates to a boolean value for each row.

Parameters:

  • df (vaex.DataFrame) – The input DataFrame.

  • filter (vaex.Expression) – A boolean Expression used to filter the DataFrame.

Returns:

A DataFrame containing only the rows that satisfy the filter.

Return type:

vaex.DataFrame

Examples

>>> import vaex
>>> from mleko.utils import get_filtered_df
>>> df = vaex.from_arrays(column1=[1, 2, 3], column2=[4, 5, 6], column3=[7, 8, 9])
>>> get_filtered_df(df, df.column1 > 1)
#    column1    column2    column3
0          2          5          8
1          3          6          9
mleko.utils.vaex_helpers.get_indices(df: vaex.DataFrame, indices: list[int]) → vaex.DataFrame#

Get DataFrame containing only the specified indices.

Parameters:

  • df (vaex.DataFrame) – The input DataFrame.

  • indices (list[int]) – A list of the indices to be extracted.

Returns:

A DataFrame containing only the specified indices.

Return type:

vaex.DataFrame

Examples

>>> import vaex
>>> from mleko.utils import get_indices
>>> df = vaex.from_arrays(column1=[1, 2, 3], column2=[4, 5, 6], column3=[7, 8, 9])
>>> get_indices(df, [0, 2])
#    column1    column2    column3
0          1          4          7
1          3          6          9
