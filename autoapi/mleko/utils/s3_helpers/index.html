download_file(destination_directory: pathlib.Path, bucket_name: str, key: str, num_workers: int = 1, multipart_threshold_gb: float = 0.5) → pathlib.Path#

Downloads a file from S3 and saves it to the destination directory.

Parameters:

  • destination_directory (pathlib.Path) – Destination directory where the file should be saved.

  • bucket_name (str) – Name of the S3 bucket.

  • key (str) – Key of the file to fetch.

  • num_workers (int) – Number of workers to use for downloading the file. Set to 1 for single-threaded download.

  • multipart_threshold_gb (float) – Threshold in GB for multipart transfer. If the file size is greater than this threshold, the file will be downloaded using multipart transfer.

Returns:

Path where the file is saved.

Return type:

pathlib.Path

upload_file(file_path: pathlib.Path, bucket_name: str, key_prefix: str, extra_args: dict[str, Any] | None = None, num_workers: int = 1, multipart_threshold_gb: float = 0.5) → S3FileManifest#

Uploads a file to S3.

Parameters:

  • file_path (pathlib.Path) – Path to the file to upload.

  • bucket_name (str) – Name of the S3 bucket.

  • key_prefix (str) – Key prefix to use for the file in the S3 bucket. The file will be uploaded to f”{key_prefix}/{file_path.name}”.

  • extra_args (dict[str, Any] | None) – Extra arguments to pass to the S3 client.

  • num_workers (int) – Number of workers to use for uploading the file. Set to 1 for single-threaded upload.

  • multipart_threshold_gb (float) – Threshold in GB for multipart transfer. If the file size is greater than this threshold, the file will be uploaded using multipart transfer.

Returns:

S3 manifest entry for the uploaded file.

Return type:

S3FileManifest

get_s3_manifest(bucket_name: str, key_prefix: str, manifest_file_name: str | None = 'manifest', file_pattern: str | list[str] = '*') → list[S3FileManifest]#

Gets the S3 manifest for the files in the S3 bucket.

Parameters:

  • bucket_name (str) – Name of the S3 bucket.

  • key_prefix (str) – Key prefix to the files in the S3 bucket.

  • manifest_file_name (str | None) – Optional name of a manifest file located on S3. If provided, the manifest from S3 will be used to determine the files to include, before applying the file pattern.

  • file_pattern (str | list[str]) – Pattern to match the files to download, e.g. *.csv or [*.csv, *.json], etc. For more information, see https://docs.python.org/3/library/fnmatch.html.

Raises:

FileNotFoundError – If no files matching the file pattern are found in the S3 bucket.

Returns:

A list of S3FileManifest objects containing the S3 keys, sizes, and last modified dates of the files in the S3 bucket.

Return type:

list[S3FileManifest]
